# WooCommerce Category Tags Filter

Ez a bővítmény automatikusan megjeleníti az adott WooCommerce termékkategóriához tartozó termékcímkéket gombok formájában a kategória archív oldalán. A gombok használatával a látogatók a címkék szerint szűrhetik a kategóriában található termékeket.

## Használat
1. Másold a bővítményt a `wp-content/plugins/` könyvtárba.
2. Aktiváld az admin felületen.
3. Nyisd meg bármelyik termékkategória oldalát: a termékcímkék automatikusan megjelennek, és kattintással szűrheted a listát.
